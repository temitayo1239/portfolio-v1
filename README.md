## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/temitayo1239/portfolio-v1.git
```

Navigate into the project

```bash
cd portfolio-v1
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open your browser and visit

```
http://localhost:3000
```
